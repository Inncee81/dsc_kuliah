Cara Membuat Thread di detikForum
- http://forum.detik.com/showthread.php?t=9

Seputar Fitur di detikForum
- memasang foto/gambar dalam thread

- mengupload foto/gambar ke dalam thread
1. dengan menggunakan
- jika menemukan pesan 'Please enter the URL of your image:' masukan url image yang telah Anda copy paste dari image hosting
2. selain itu juga bisa dengan menggunakan
- browse foto/ image di dalam komputer/ laptop Anda ATAU
- copy paste url image dari image hosting
- setelah file di-attach klik attachment sekali lagi dan pilih file pilihan Anda. Pastikan besaran image yang Anda attach sesuai dengan ketentuan

- membuat spoiler
- memasang/mengedit avatar
- membuat polling dalam thread
- mengganti password di detikForum/detikID

Bagaimana mengganti email detikForum?
Quote:
kirim email ke info[at]detik.com, sertakan username, email yang dipakai saat ini dan email pengganti di dalamnya

Dengan cara men-subscribe thread favorit Anda. untuk penggunaannya sila ikuti langkah-langkah berikut:
a. klik thread yang ingin di-subscribe

Social Groups
Merupakan fitur untuk interaksi antar member dalam sebuah grup. Member dapat membuat grup, meng-invite teman, membuat topik ataupun mengirimkan foto ke dalam grup.
Quote:
CREATE GROUP
klik social groups di navbar atas
pilih create a new group
pilih kategori
isi group name
isi group description
pilih group type
- public
- moderate
- invite only
pilih group options
- enable albums
- enable messages
klik save

JOIN GROUP
ke social group
pilih group yang diinginkan
pilih 'join group'
'Are you sure you want to join the group called "xxxxxxx"?' klik 'yes'

MENINGGALKAN GROUP
dalam posting klik 'leave group'
Membuat Tags
Quote:
klik new thread
isi message box
isi tags
- tag maksimal 5 per thread
- karakter dalam tag minimal 4 maksimal 10
- pisahkan tag dengan koma
- dilarang memasukan forbidden words dalam tag
klik 'save changes'
Seputar Visitor Message di profile Anda
Quote:
Visitor Message berfungsi mirip wall di social network.
- cara mengaktifkan usercp --> edit options --> enable visitor messaging --> save changes
- untuk melihat visitor message klik 'your profile' di usercp
User Albums
Quote:
ADD ALBUMS
isi judul album
isi 'description'
pilih 'click here to upload pictures!'
browse foto yang diinginkan (maksimal tiga foto per upload dengan size yang telah tertera dalam ketentuan)
klik upload pictures

MENARUH PIC DI GROUP ALBUMS
klik kotak kecil di setiap pictures
pilih 'add selected pictures to'
pilih group tujuan
add pictures
Customize Profile
Quote:
usercp --> customize profile
note: apabila ingin mempergunakan background image, gunakan image yang terdapat dalam album Anda, copy paste url-nya
Social Bookmarking
Quote:
ke thread yang mau di-bookmark --> pilih social networking --> apabila Anda berada dalam posisi login di social networking tersebut (FB, Twitter) link thread secara otomatis akan berada dalam pesan yang akan diposting
Memasang youtube di DF

- ambil url youtube, misalnya: http://www.youtube.com/watch?v=RUnHM5ii014
- masukan kode youtube
